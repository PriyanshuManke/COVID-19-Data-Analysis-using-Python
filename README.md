# COVID-19 Data Analysis using Python

This project focuses on analyzing global COVID-19 case data to uncover trends, patterns, and insights. It includes complete data cleaning, transformation, and exploratory data analysis (EDA) using Python.

---

## Project Overview
The analysis covers:
- Time-series trends of confirmed and recovered cases  
- Country-wise spread patterns  
- Growth rate and peak analysis  
- Data cleaning & preprocessing  
- Visualizations for understanding the pandemic progression  

The project helps identify how COVID-19 evolved across different regions and time periods.

---

## Technologies & Libraries Used
- **Python**
- **Pandas** (data cleaning & manipulation)
- **NumPy**
- **Matplotlib** (visualization)
- **Jupyter Notebook**

---

## Key Insights
- Countries show different infection peaks based on geographical and population factors.
- Recovery trends reveal improvement periods across regions.
- Visual patterns highlight the spread rate over time.
- Time-series visualizations help understand daily/weekly fluctuations.
